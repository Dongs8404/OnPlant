# on-plant-ai

AI 기반 반려 식물 로봇 프로젝트입니다.  
센서 데이터를 기반으로 식물 상태를 분석하고, 사용자에게 시각적으로 상태와 관리 방법을 알려주는 Flask 기반 MVP입니다.

## 현재 구현 상태

- 랜덤 센서값 기반 데모
- 토양 수분, 온도, 습도, 조도 상태 판단
- 식물 상태별 이미지 출력
- 관리 추천 메시지 출력
- Flask 웹 화면 제공
